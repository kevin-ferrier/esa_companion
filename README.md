# ESA Marathon Extension

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run build-watch
```
You can them go to your chromium browser, activate `developer mode` and select `load unpacked` and locate the dist folder.

## Libs
- VueJS
- Tailwind CSS
